# OpenCode Start Here

This repository is prepared for OpenCode continuation across future chats.

Before doing any work:

1. Read [AGENTS.md](AGENTS.md).
2. Read [OPEN_CODE_HANDOFF.md](OPEN_CODE_HANDOFF.md).
3. Run:

```bash
git status --short --branch
git diff
```

Then continue from the `Next Step` section in [OPEN_CODE_HANDOFF.md](OPEN_CODE_HANDOFF.md), unless the user gives a newer instruction.

Before stopping, update [OPEN_CODE_HANDOFF.md](OPEN_CODE_HANDOFF.md) with:

- current branch
- active task
- changed files
- commands run
- verification status
- blockers
- next exact step
